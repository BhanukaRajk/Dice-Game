******************** Game Instructions ************************

> Players can roll two dice in one chance.

> If some player got the same values from the two dices(except 1), he/she will get another bonus chance.

> Any player who gets the value 1 in both two dices has to score again from 0.

> Any player who firstly gets exactly 100 is the winner.

> If any player who got total points as 98 or 99, has to get the value 1 in both two dices and need to score again from 0.

> After a single round ended, players can play using the same names by replay or can play a new game by pressing new game.
